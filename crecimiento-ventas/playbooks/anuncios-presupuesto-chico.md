# 💸 Playbook: Anuncios con presupuesto chico

**Objetivo:** sacarle el máximo a poca plata. Con presupuesto limitado, **no** salgas a "buscar gente
nueva en frío" (caro). Apuntá a quien ya mostró interés: ahí cada peso rinde 5–10×.

## Orden de prioridad (de más a menos rentable)

1. **Retargeting** (lo #1 con poca plata) ⭐
   - Anuncio a quien **ya visitó la tienda / vio un producto / dejó el carrito** y no compró.
   - Requiere el **píxel de Meta** (Instagram/Facebook) instalado en la web. Es lo primero a configurar.
   - Mensaje: el producto que miró + un empujón ("seguís a tiempo", envío gratis, cupón).

2. **Catálogo dinámico (Advantage+ / catálogo)** ⭐
   - Conectás el catálogo y Meta muestra automáticamente los productos a quien los vio. Escala solo.

3. **Retargeting a tu lista** (clientes/prospectos)
   - Subís tu base (emails/teléfonos) como **público personalizado** y le mostrás anuncios. Y un
     **público similar (lookalike)** a partir de tus mejores clientes para buscar parecidos.

4. **Google: Search de marca + Performance Max acotado**
   - Aparecé cuando buscan "El Pasaje Dental" o tus productos. Barato y de alta intención.

## Reglas para no quemar plata
- **Empezá chico** ($/día bajo), 1 objetivo, medí 7–14 días, **doblá lo que funciona**, cortá lo que no.
- **1 público + 1 mensaje + 1 producto/categoría** por anuncio (no mezclar).
- **Creativo nativo** (que parezca contenido, no aviso): reels/fotos reales de producto.
- **Llevá a una página clara** (la categoría/producto, no el home genérico).
- **Medí ventas, no likes.** Cupón propio del anuncio (`ADS10`) + el píxel para atribuir.

## Pre-requisitos técnicos (configurar una vez)
- **Píxel de Meta** + **API de conversiones** en la tienda. (Pedímelo y vemos de instalarlo.)
- **Catálogo** sincronizado a Meta/Google (feed de productos).
- Cuenta de **Google Ads** + **Merchant Center** para Shopping.

## Con $ chico, el plan mínimo
Píxel instalado → **retargeting + catálogo dinámico** en Meta + **Search de marca** en Google.
Eso solo ya recupera ventas que hoy se pierden, con muy poca inversión.
